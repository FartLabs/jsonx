# jsonx

[![deno doc](https://doc.deno.land/badge.svg)](https://deno.land/x/jsonx)

JSX runtime for composing JSON data.

## Development

### Style

Run `deno fmt` to format the code.

Run `deno lint` to lint the code.

### Testing

Run `deno test` to run the unit tests.
